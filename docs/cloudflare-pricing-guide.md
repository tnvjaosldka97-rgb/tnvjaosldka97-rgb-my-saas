## 요약 및 핵심 결론

### 핵심 요약

| 항목 | Cloudflare | AWS Lambda | Google Cloud | Azure |
| --- | --- | --- | --- | --- |
| **서버리스 무제한 대역폭** | ✅ 무제한 | ❌ $0.09/GB | ❌ 유료 | ❌ 유료 |
| **Free Tier 요청수** | 10만/일 | 100만/월 | 200만/월 | 100만/월 |
| **Cold Start** | 없음 (1ms 미만) | 100-1000ms | 100-400ms | 있음 |
| **CPU 과금 방식** | CPU 시간만 | 전체 지속 시간 | 리소스 기반 | 리소스 기반 |
| **I/O 대기 시간 과금** | ❌ 없음 | ✅ 있음 | ✅ 있음 | ✅ 있음 |

### 주요 장점

1. **무제한 무제한 대역폭** - Free/Paid 모두 이그레스 비용 없음
2. **CPU 시간 기반 과금** - I/O 대기 시간에 과금되지 않음 (70-90% 비용 절감)
3. **Zero Cold Start** - V8 Isolate 아키텍처로 즉시 실행
4. **통합 생태계** - CDN + Workers + Storage + AI를 하나의 플랫폼에서

### 주의사항

1. **Worker당 128MB 메모리 제한**
2. **Free Tier 10만 요청/일 한도**
3. **Node.js 완전 호환 불가** (일부 API 제한)

| 기능 | Free | Paid |
| --- | --- | --- |
| **쿼리 차원수** | 3,000만/월 | 5,000만/월 포함&lt;br&gt;+$0.01/백만 |
| **저장 차원수** | 500만 | 1,000만 포함&lt;br&gt;+$0.05/1억 |

**비용 계산 공식**:

```
비용 = (쿼리 벡터 + 저장 벡터) × 차원 × ($0.01 / 1,000,000)
     + (저장 벡터 × 차원 × $0.05 / 100,000,000)
```

**예시 비용** (768차원, 5만 벡터, 월 20만 쿼리):

- 총 비용: 약 $1.94/월 (Paid 플랜 포함분 내)

## 8. 비용 최적화 전략

### 8.1 Workers 최적화

1. **CPU 시간 최소화**

   - I/O 작업은 CPU 시간에 포함되지 않음 (네트워크 요청 최대한 활용)
   - 무거운 계산은 클라이언트로 오프로드

2. **캐싱 전략**

   - Cache API 활용으로 반복 요청 감소
   - KV로 자주 접근하는 데이터 저장

3. **지능적 라우팅**

   - 정적 자산은 ASSETS 바인딩 사용 (요청 과금 제외)

### 8.2 데이터베이스 최적화

1. **D1 인덱싱**

   - 적절한 인덱스로 rows_read 감소
   - 쿼리 계획 확인

2. **배치 처리**

   - 여러 쿼리를 트랜잭션으로 묶기

3. **KV 캐싱**

   - 자주 읽는 데이터는 KV에 캐시

### 8.3 스토리지 최적화

1. **R2 Infrequent Access**

   - 자주 접근하지 않는 데이터는 IA 저장

2. **Class B 작업 선호**

   - GET이 PUT보다 훨씬 저렴
